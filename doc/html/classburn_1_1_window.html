bool burn::Window::isContextCreated ( )
inlinestatic

This function is used to ensure, that an OpenGL-Context exists. Calling OpenGL-Methods will result in crashes when no context is created. This is internally used by all classes that call OpenGL-Methods.

A Context is created as soon as a window was created.
